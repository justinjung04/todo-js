Introduction

Developers these days have different options of JavaScript frameworks to choose from. They include:

  • Backbone.js
  • AngularJS
  • Ember.js
  • KockoutJS
  • Dojo
  • Agility.js

and many more.

Each framework provides great tools to make high-quality code. However, it is critical to understand each one's pros and cons to make correct decisions for different projects.

To demonstrate and compare different frameworks, I have created TodoJS - a project which offers the same Todo-application implemented using different JavaScript frameworks that is available today.
